# GreenBin – Projet DWWM

Projet réalisé dans le cadre du titre professionnel  
**Développeur Web et Web Mobile (DWWM)**.

## Objectif du projet
GreenBin est une application web de gestion et de suivi des déchets,
développée selon une architecture full-stack.

## Contenu du dépôt
- backend : API Node.js / Express
- frontend : application React
- database : scripts SQL
- dossier-projet : Dossier Professionnel (DP)
- support-presentation : support de présentation du projet
